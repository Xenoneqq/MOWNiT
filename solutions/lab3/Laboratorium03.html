W celu stworzenia macierzy Vandermondea wybrano cztery wzory. Zostay one zaimplementowane jako warto zwracana przez funkcj equation gdzie n okrela obecnie uywan funkcj.

def equation(t, j, n):
    match n:
        case 0:
            return t**(j-1)
        case 1:
            return (t - 1900)**(j-1)
        case 2:
            return (t-1940)**(j-1)
        case 3:
            return ((t-1940)/40)**(j-1)
    return 1

Funkcja create_matrix tworzy macierz Vandermondea korzystajc z wybranej funkcji dla kadego z punktw X

In[78]:
def create_matrix(X, n):
    matrix = np.zeros((len(X), len(X)))
    for i in range(len(X)):
        for j in range(len(X)):
            matrix[i, j] = equation(X[i], j+1, n)
    return matrix

Z czterech stworzonych macierzy zostaa wybrana tylko jedna, miaa ona najniszy wspczynnik uwarunkowania, ktry okrela podatno na bdy numeryczne. Nisze wspczynniki wykazuj stosunkowo maa podatno na bdy.

C - Rysowanie wykresu z wartociami Interpolowanymi

W celu narysowania wykresu stworzona zostaa funkcja interpolate wykorzystujc schemat Hornera do zwrcenia przyblionej wartoci argumentu x. Funkcja obejmuje wartoci z poza dziedziny.

In[80]:
def interpolate(t, coeff, degree=0):
    if degree == len(coeff) - 2:
        return coeff[degree] + coeff[degree+1] * ((t-1940)/40)
    return coeff[degree] + ((t-1940)/40) * interpolate(t, coeff, degree+1)

E - Interpolacja metod Lagrange'a

Ta cz zadania ma na celu wyznaczenie corocznych wartoci za pomoc wielomianu interpolacyjnego Lagrange'a. W tym celu zostay stworzone funkcje lagrange_basis oraz lagrange_interpolation.

In[83]:
def lagrange_basis(x, i, t):
    result = 1.0
    for j in range(len(x)):
        if j != i:
            result *= (t - x[j]) / (x[i] - x[j])
    return result

def lagrange_interpolation(x, y, t):
    result = 0.0
    for i in range(len(x)):
        result += y[i] * lagrange_basis(x, i, t)
    return result

F - Interpolacja metod Newtona

Ta cz zadania ma na celu wyznaczenie corocznych wartoci za pomoc wielomianu interpolacyjnego Newtona. W tym celu zostay stworzone funkcje divided_differences oraz newton_interpolation.

In[85]:
def divided_differences(x, y):
    n = len(x)
    coef = np.array(y, dtype=float)
    for j in range(1, n):
        for i in range(n-1, j-1, -1):
            coef[i] = (coef[i] - coef[i-1]) / (x[i] - x[i-j])
    return coef

def newton_interpolation(x, coef, t):
    n = len(coef)
    result = coef[-1]
    for i in range(n-2, -1, -1):
        result = result * (t - x[i]) + coef[i]
    return result

Wniosek

Interpolacja to wietne narzdzie pozwalajce na przyblienie wartoci wychodzcych poza znane nam argumenty. Metoda ta nie powinna by jednak uywana do przewidywania wynikw zdarze przyszych jej najlepsze zastosowanie znajduje si w uzupenianiu danych z przeszoci.

Majc n punktw, moemy skonstruowa dokadnie jeden wielomian interpolacyjny stopnia n-1, ktry przechodzi przez wszystkie te punkty i odwzorowuje rzeczywiste wartoci. Cho ten wielomian jest unikalny, moe by przedstawiony na rne sposoby w zalenoci od wybranej bazy funkcji, np. bazy Newtona czy bazy Lagrangea co daje rne formy jego implementacji.
